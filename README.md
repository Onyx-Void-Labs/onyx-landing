# Onyx Landing Page

The official landing page for **Onyx** (onyxvoid.com), a secure, end-to-end encrypted markdown editor.

## Features
- **Modern Performance**: Built with zero dependencies for instant loading.
- **Glassmorphic Design**: Clean, dark-mode aesthetic with interactive glow elements.
- **High CTA**: Direct links to GitHub Releases for seamless downloads.

## Deployment
This site is designed to be hosted on **Cloudflare Pages**.
1. Push this repository to GitHub.
2. Link your GitHub account to Cloudflare Pages.
3. Select this repo and deploy.
